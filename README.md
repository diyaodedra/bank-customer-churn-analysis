# Banking Customer Churn Analysis

## Overview
This project analyses customer churn data from a banking dataset to identify churn patterns, high-risk customer segments, and actionable retention opportunities.

## Objectives
- Calculate overall customer churn rate
- Identify churn patterns across customer demographics and behaviour
- Segment churn by age, tenure, and activity status
- Provide data-driven business recommendations to reduce churn

## Tools Used
- Microsoft Excel
  - Pivot Tables
  - KPI calculations
  - Data segmentation
  - Dashboard visualisation

## Key KPIs
- Total Customers: 10,000
- Churned Customers: 2,037
- Churn Rate: 20.37%

## Business Recommendations
- Strengthen onboarding and engagement during the first three years
- Implement targeted retention strategies for older customer segments
- Increase engagement initiatives for inactive customers to reduce churn risk
